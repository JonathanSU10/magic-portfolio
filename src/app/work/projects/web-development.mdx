## Overview

Over the course of academic and internship projects, I developed several web-based information systems tailored to administrative and operational needs. These systems are backend-focused and are designed to handle business logic reliably.

## Key Focus Areas

- **Backend Logic & CRUD Reliability**: Ensuring that all data operations are secure, efficient, and consistent.
- **Data Validation**: Enforcing strict validation rules to maintain data integrity across operational dashboards.
- **Workflow Implementation**: Automating business processes through systematic web interfaces, reducing manual errors.
- **UI Structuring**: Building clean, intuitive admin dashboards that prioritize usability and quick access to information.

## Technologies Used

- **PHP**
- **TypeScript**
- **React**
- **CodeIgniter**
- **Laravel**

These technologies form the core of the web applications I build, combining stable backend frameworks with dynamic frontend interfaces for comprehensive information systems.
